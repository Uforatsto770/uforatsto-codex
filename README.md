# UFORATSTO INC MVP Landing Page

Simple static MVP landing page for `uforatsto.com`.

## Structure

- `index.html` - page markup and business content
- `css/styles.css` - responsive layout and visual styling
- `js/main.js` - mobile navigation toggle and small UI behavior

## Purpose

This version is designed to replace a parking page with a credible one-page business website focused on:

- trade and logistics support
- operational coordination
- mushroom growing equipment
- grow tents in different sizes
- small, medium, and industrial setups
- climate control and automation
- turnkey solutions

## Local Preview

Open `index.html` in a browser.

## Deployment

Upload the repository to GitHub and enable GitHub Pages from the root of the default branch.
